
# General Info
This app is clone app of githun trending page

# Used Stack

Created by `Create React App` with Typescript

- React.js
- TypeScript
- Nextjs


# Folder Structure

```
/src
  ├── components
  ├── hooks
  ├── screens
  ├── types
  ├── util               # Utilites for hold helper functions 
  
```

# Setup

Install dependencies:

```sh
yarn
```

Run App

```sh
yarn dev
```
